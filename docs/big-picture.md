# 全体像
NerconeOSは大規模なプロジェクトであり、複数の要素によって成り立っています。

- カーネルに最低限必要な機能のみを持つ「NerKernel」
- NerKernelにない多くの重要な機能を追加するための「サブモジュール(NKMod)」「カーネル機能拡張(NKExt)」
- ソフトウェアを実行するための「NeRunE<!-- RunERunE --> (Nercone Runtime Environment)」
- NerconeOS向けソフトウェアの開発環境や既存の言語向けのライブラリなどをまとめた「NerconeOS SDK」
- Linux向けソフトウェアを実行するための「LRFN」
- Windowsアプリを実行するための「WRFN」
- Androidアプリを実行するための「ARFN」
- 直感的で簡単に操作するためのデスクトップ環境「Nesktop」
- Nesktop上でキーボードを使用して文章を入力するための入力システムとIME「Ninput」
